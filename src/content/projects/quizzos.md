## Le Projet

Application web de quiz moderne et immersive, offrant une expérience utilisateur unique avec des animations fluides et un design distinctif. Le projet met l'accent sur la modularité, les performances et l'engagement utilisateur à travers une interface ludique et intuitive.

## Architecture & Technologies

### Frontend Moderne

- **Architecture modulaire** avec routing client-side
- **Système de composants** réutilisables
- **Gestion d'état** via localStorage

### Interface Utilisateur

- **Design System** cohérent et moderne
- **Animations fluides** avec GSAP
- **Responsive Design** complet
- **SCSS modulaire** avec architecture 7-1
- **Composants réutilisables**

### Build & Performance

- **Vite.js** pour le bundling optimisé
- **Modules ES6** pour une meilleure maintenabilité
- **Assets optimisés** pour les performances
- **Code splitting** automatique
- **Hot Module Replacement**
